# How To Run

1. Run R
2. Load the script into your environment 
`   source("run_analysis.R")   `
3. Run analysis function 
`   run_analysis()   `
4. It will produce simultaneously
  1. tidy data in your environment
  2. tidy data in tidy_data.csv file
